Header

El header es una etiqueta de contenedor semántico que se utiliza para agrupar contenido introductorio o de navegación de una página web o de una sección específica de una página. Normalmente, contiene elementos como el logo, el título, la barra de navegación, y puede incluir otros elementos como un buscador o un eslogan. La etiqueta header ayuda a estructurar el contenido y mejorar la accesibilidad de la página.

Footer

El footer es una etiqueta semántica que se utiliza para definir el pie de página de un documento o de una sección específica dentro de una página. Normalmente, el footer incluye información de contacto, derechos de autor, enlaces a políticas de privacidad, enlaces adicionales o cualquier otro contenido relevante para el final de una página o sección.

Color de la web

El color en la web se refiere al uso de valores de color para dar estilo y vida a los elementos de una página. Se puede aplicar color al texto, fondos, bordes y otros elementos, usando valores de color que se definen mediante CSS.

Especificidad

La especificidad es una regla que determina qué estilo se aplica a un elemento cuando hay varias reglas en conflicto que afectan al mismo elemento. Específicamente, la especificidad ayuda a decidir qué estilo tiene prioridad y, por lo tanto, cuál se aplicará en último término.

Tablas

las tablas son estructuras que permiten organizar y mostrar datos en filas y columnas. Se crean con la etiqueta table y se componen de varias etiquetas para definir la estructura interna, como tr para las filas, th para las celdas de encabezado, y td para las celdas de datos.

Formularios CSS

los formularios son elementos que permiten a los usuarios ingresar y enviar datos al servidor. Los formularios se crean con la etiqueta form y contienen varios elementos de entrada, como campos de texto, botones de selección, casillas de verificación, y botones de envío. Aunque HTML crea la estructura y funcionalidad básica de los formularios, CSS se utiliza para darle estilo y mejorar la apariencia visual de estos elementos, mejorando la experiencia del usuario.
